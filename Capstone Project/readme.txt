# Travel Agency Website

## Project Title & Theme
**Travel Agency Website** – A responsive website showcasing travel packages, destinations, and services offered by a travel agency.  
Theme focuses on **modern, clean, and visually appealing design** for travelers.

## Sections Implemented
- **Hero Section:** Large banner image with call-to-action button.  
- **Navigation Bar:** Links to Home, About, Packages, and Contact sections.  
- **About Section:** Brief description about the agency.  
- **Packages Section:** Travel destinations displayed as cards with images and short details.  
- **Contact Section:** Form to allow users to send inquiries.  
- **Footer:** Basic footer with social links and copyright information.  

## Layout Techniques & Breakpoints
- **Layout Techniques:**  
  - Flexbox for layout alignment and responsive sections.  
  - Grid for packages/cards display.  
  - Media queries for responsiveness.  

- **Breakpoints Used:**  
  - `@media (max-width: 1200px)` → Adjust large desktop layout  
  - `@media (max-width: 768px)` → Tablet view  
  - `@media (max-width: 480px)` → Mobile view, stack content vertically
